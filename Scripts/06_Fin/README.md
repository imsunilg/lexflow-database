# 06_Fin

Billing, trust accounting and time tracking domain: rate cards, invoices, payments, credit notes, dunning, trust ledger, tax configs, number series, time entries. Highest-risk schema — immutability and balance-integrity triggers live here. Maps to PRD §18 fin.* tables, Modules 8 and 9.

DB-6 built 19 of the 22 objects (all except InvoiceStatusHistory, TrustReconciliations, TrustReconciliationItems, which weren't part of that prompt's scope and remain for a future prompt).
